Performance

Hardware acceleration with force3D

Submenu Support

Nested navigation with proper animations

Animation Flow

  1. Menu Open:
    • Button text rotates out (rotationX: 90, yPercent: 100)
    • Container slides in from left (xPercent: -100 to 0)
    • Menu items stagger in from top (yPercent: -100 to 0)
    • Social and contact links appear last
  2. Menu Close:
    • Reverse animations with different timing
    • Container slides out (xPercent: 0 to -100)
    • Button text rotates back in

Integration Steps

  1. Include GSAP library in your HTML head
  2. Add the normandpllc-menu.css stylesheet
  3. Copy the navigation HTML structure
  4. Include normandpllc-menu.js script
  5. Customize colors and fonts in CSS variables
